Pygin

Very basic demo of Python adapter.

Installation:
Put x64\Debug\pygin.dll or Win32\Debug\pygin.dll to %FARHOME%\Adapters.
Put plugins\helloworld directory to %FARHOME%\Plugins.
Make sure you have Python 3 (e.g. C:\Program Files\Python35 or C:\Program Files (x86)\Python35-32) directory in the %PATH%.

Usage:
F11 - Hello Python world - Ctrl-O
